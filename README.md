## 1. О дисциплине

**Цель дисциплины.** Сформировать у студентов систематизированное представление о современных архитектурах нейронных сетей для компьютерного зрения и выработать практические навыки их применения для решения реальных задач анализа изображений и видео.

**Основные задачи.**
1. Изучить архитектуры нейронных сетей для компьютерного зрения: от классических методов до современных свёрточных сетей, трансформеров и генеративных моделей.
2. Освоить математические основы работы нейросетевых моделей и методы их оптимизации и регуляризации.
3. Сформировать практические навыки реализации и обучения моделей для детекции, сегментации, генерации и мультимодального анализа изображений.
4. Научить работать с инструментами и библиотеками для обработки данных, обучения моделей и оценки их качества.
5. Развить умение адаптировать и применять модели для решения прикладных задач с учётом ограничений.

**Структура и содержание.** Дисциплина состоит из 4 модулей (объём 144 часа / 4 зач. ед.):
- Модуль 1. Фундаментальные архитектуры и методы глубокого обучения для зрения (20 ч)
- Модуль 2. Основные задачи компьютерного зрения: детекция, сегментация, генерация (28 ч)
- Модуль 3. Современные архитектуры: внимание, трансформеры и метрическое обучение (20 ч)
- Модуль 4. Продвинутые и гибридные методы (31 ч)

Обучение включает лекции (32 ч), лабораторные работы (48 ч), курсовой проект и завершается экзаменом.

**Пререквизиты и связи с другими дисциплинами.** Дисциплины: «Технологии компьютерного зрения», «Современные методы машинного обучения», «Нейросетевые технологии», «Интеллектуальные методы оптимизации».

**Планируемые результаты обучения.** По завершении дисциплины обучающийся сможет:
1. Применять классические и современные архитектуры CNN для задач анализа изображений.
2. Разрабатывать и обучать модели для детекции объектов, сегментации и генерации изображений.
3. Использовать механизмы внимания и Vision Transformer в задачах классификации и верификации.
4. Реализовывать мультимодальные решения и системы распознавания текста.
5. Решать сквозные прикладные задачи компьютерного зрения.

Полное описание приведено в [РПД](docs/rpd.md).

## 7. Порядок заполнения

1. Заполните РПД и сформулируйте проверяемые результаты обучения.
2. Выберите релевантные профессиональные роли и компетенции КРМ.
3. Для каждого результата выберите индикатор и требуемый уровень.
4. Заполните модель измерения.
5. Создайте КИМ и рубрики оценивания по шаблонам.
6. Согласуйте систему итогового балла.
7. Добавьте ресурсы, сведения о команде и лицензию.
