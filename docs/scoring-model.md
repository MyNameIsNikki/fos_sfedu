# Модель оценивания

## 100-балльная шкала

| Компонент | Макс. балл (в 100-б. шкале) |
|---|---|
| Лабораторные работы (9 шт.) | 60 |
| Экзамен (индивидуальный кейс) | 40 |
| Курсовой проект (альтернатива экзамену) | 40 |

Итог = Лаб + Экзамен (или Проект). Максимум — 100 баллов.

## Формула нормализации

Для каждой лабораторной работы:

```
Балл_в_100 = Оценка_студента_из_КИМ / Локальный_максимум × Вес_в_60
```

Пример: студент получил 15/20 за ЛР-1 → 15 ÷ 20 × 5 = 3,75 балла в 100-балльной шкале.
