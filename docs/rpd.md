# Рабочая программа дисциплины

## 1. Общие сведения

| Поле | Значение |
|---|---|
| Название дисциплины | Глубокое обучение для зрения |
| Образовательная программа | [ЗАПОЛНИТЬ] |
| Уровень образования | магистратура |
| Объем | 64 академических часа |
| Формат обучения | очный |
| Форма промежуточной аттестации | экзамен (индивидуальный кейс) |

## 2. Цель и задачи

**Цель:** Формирование у обучающихся систематических знаний и практических навыков в области глубокого обучения для задач компьютерного зрения: от классических методов детекции и сегментации до современных архитектур на основе трансформеров и мультимодальных моделей.

**Задачи:**

1. Изучить фундаментальные архитектуры свёрточных нейронных сетей и методы глубокого обучения для анализа изображений.
2. Освоить современные подходы к детекции объектов, семантической сегментации и генерации изображений.
3. Овладеть методами метрического обучения, механизмами внимания и архитектурами на основе трансформеров.
4. Получить практические навыки решения прикладных задач CV: мультимодальное описание, распознавание текста, 3D-зрение.

## 5. Система оценки результатов обучения

Максимальная сумма баллов за семестр — 100. Распределение:

| Вид работы | Баллы |
|---|---|
| Лабораторные работы (9 шт.) | 45–65 |
| Экзамен (индивидуальный кейс) | 30–40 |
| Дополнительные задания (продвинутый уровень) | 10–20 |

**Шкала итоговой оценки:**

| Диапазон баллов | Оценка |
|---|---|
| 0–50 | неудовлетворительно |
| 51–70 | удовлетворительно |
| 71–89 | хорошо |
| 90–100 | отлично |
