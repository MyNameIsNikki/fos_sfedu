# LLM и промпты

## Рекомендуемые промпты

### Для объяснения кода
```
Объясни, что делает следующий код на PyTorch:
[код]
```

### Для отладки
```
У меня ошибка в PyTorch: [ошибка]. 
Код: [код]. Как исправить?
```

### Для генерации шаблонов
```
Создай шаблон класса Dataset для PyTorch, загружающего изображения из папки.
```

## Ограничения
- LLM не должен использоваться для полного решения заданий
- Студент должен понимать и уметь объяснить сгенерированный код
- Все результаты работы LLM должны быть проверены студентом
