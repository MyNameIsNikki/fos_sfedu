# КИМ-3.1. Лабораторная работа №6. Внедрение и использование Vision Transformer (ViT)

## Назначение
Проверка способности применять архитектуры на основе механизма внимания и Vision Transformer для классификации изображений.

## Проверяемые результаты (дескрипторы)
- Применяет предобученный Vision Transformer для классификации изображений
- Сравнивает ViT с CNN архитектурами
- Настраивает параметры fine-tuning
- Анализирует влияние размера патчей на качество
- Разрабатывает и отлаживает прикладные решения на Python с использованием HuggingFace Transformers, PyTorch (PL-1.1)

## Условия выполнения
- Среда: Python 3.8+, Jupyter Notebook
- Библиотеки: PyTorch, torchvision, transformers (HuggingFace), timm, matplotlib, captum

## Материалы задания
- **Датасет:** CSU101 или подмножество ImageNet (100 классов)

### Задание
1. Загрузите предобученный ViT-Base из HuggingFace transformers или timm
2. Замените classification head под свой датасет
3. Дообучите ViT с разными learning rate (1e-4, 1e-5, 1e-6)
4. Визуализируйте карты внимания для нескольких слоёв (self-attention maps)
5. Проанализируйте, как меняется глобальный контекст на разных уровнях
6. Сравните ViT с ResNet-50 по точности и скорости инференса
7. Экспериментируйте с размером патчей (patch_size=16 vs 32)
8. Постройте confusion matrix и проанализируйте ошибки

## Формат сдачи
Отчёт с визуализациями карт внимания, сравнением ViT vs CNN, анализом влияния гиперпараметров.
