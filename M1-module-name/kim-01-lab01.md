# КИМ-1.1. Лабораторная работа №1. Классическое детектирование объектов по Виоле-Джонса

## Назначение
Проверка способности применять классические методы компьютерного зрения для детектирования объектов на изображениях.

## Проверяемые результаты (дескрипторы)
- Применяет метод Виолы-Джонса для детектирования объектов на изображениях
- Использует библиотеку OpenCV для работы с каскадными классификаторами
- Настраивает каскадные классификаторы на собственном датасете
- Оценивает качество детекции (TPR, FPR, Precision, Recall)

## Условия выполнения
- Самостоятельная работа
- Среда: Python 3.8+, Jupyter Notebook
- Библиотеки: OpenCV, NumPy, scikit-learn, matplotlib

## Материалы задания
- **Датасет:** FDDB или Caltech Pedestrians (небольшой набор для детекции лиц/пешеходов)
- **Шаблон отчёта:** Jupyter Notebook

### Задание (2 этапа)

**Этап 1. Каскад Виолы-Джонса**
- Загрузите предобученный каскад Хаара из OpenCV (`haarcascade_frontalface_default.xml`)
- Обучите каскад на своём датасете (добавьте свои положительные и отрицательные выборки)
- Для обучения используйте утилиты OpenCV (`opencv_createsamples`, `opencv_traincascade`)
- Оцените точность (TPR, FPR) и скорость детекции (FPS)
- Визуализируйте результаты на нескольких тестовых изображениях

**Этап 2. Дескрипторы HOG + SVM**
- Извлеките HOG-дескрипторы из изображений (`hog.compute()`)
- Обучите линейный SVM (`sklearn.svm.LinearSVC`) для классификации объектов (лицо/не лицо)
- Оцените mAP
- Сравните качество детекции с каскадом Виолы-Джонса по метрике mAP
- Проанализируйте, в каких случаях каждый метод даёт преимущество
