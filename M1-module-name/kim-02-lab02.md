# КИМ-1.2. Лабораторная работа №2. Сравнительный анализ свёрточных архитектур

## Назначение
Проверка способности реализовывать, дообучать и сравнивать современные свёрточные архитектуры для классификации изображений.

## Проверяемые результаты (дескрипторы)
- Реализует и сравнивает многослойный перцептрон и базовые свёрточные сети (LeNet, AlexNet)
- Применяет предобученные модели (ResNet) и дообучает их под задачу классификации
- Модифицирует архитектуру (SE block, ECA block, inverted bottleneck)
- Анализирует влияние глубины и регуляризации на точность
- Разрабатывает и отлаживает прикладные решения на Python с использованием PyTorch, torchvision (PL-1.1)

## Условия выполнения
- Самостоятельная работа
- Среда: Python 3.8+, Jupyter Notebook
- Библиотеки: PyTorch, torchvision, NumPy, matplotlib

## Материалы задания
- **Датасет:** CIFAR-10 или CSU101
- **Шаблон отчёта:** Jupyter Notebook

### Задание (3 этапа)

**Этап 1. Базовое воспроизведение**
- Загрузите предобученный ResNet50 из torchvision.models
- Замените последний полносвязный слой под размер датасета
- Дообучите с небольшим learning rate (1e-4)
- Зафиксируйте accuracy и FPS

**Этап 2. Кастомизация блоков (SE block, ECA block)**
- Реализуйте SE block: global average pooling → два FC слоя → сигмоида
- Реализуйте ECA block: global average pooling → 1D convolution → сигмоида
- Встройте блоки после каждого bottleneck в ResNet
- Обучите обе модификации, сравните точность и количество параметров
- Визуализируйте карты активации

**Этап 3. Inverted bottleneck (MobileNetV2)**
- Реализуйте блок: расширение каналов (expand_ratio=3, 6, 12) → depthwise conv → проекция
- Соберите последовательность блоков с разными expand_ratio
- Постройте график зависимости accuracy от числа параметров
- Сделайте вывод об оптимальном коэффициенте
