# КИМ-2.3. Лабораторная работа №5. Генерация изображений с помощью GAN

## Назначение
Проверка способности реализовывать и обучать генеративные состязательные сети для синтеза изображений.

## Проверяемые результаты (дескрипторы)
- Реализует генеративно-состязательную сеть (DCGAN)
- Анализирует стабильность обучения GAN
- Применяет методы борьбы с коллапсом мод
- Оценивает качество генерации (FID)

## Условия выполнения
- Среда: Python 3.8+, Jupyter Notebook
- Библиотеки: PyTorch, torchvision, NumPy, matplotlib, scipy

## Материалы задания
- **Датасет:** CelebA, Anime Faces или CIFAR-10

### Задание
1. Загрузите датасет изображений и создайте DataLoader
2. Реализуйте архитектуры Generator и Discriminator для DCGAN
3. Реализуйте цикл обучения GAN с фиксацией loss
4. Визуализируйте генерируемые изображения в процессе обучения (каждые N эпох)
5. Примените методы стабилизации: label smoothing, BatchNorm, LeakyReLU
6. Оцените качество генерации с помощью FID (Fréchet Inception Distance)
7. Экспериментируйте с размером latent space, learning rate
8. Сделайте вывод о влиянии гиперпараметров на качество

## Формат сдачи
Отчёт с динамикой обучения, примерами генерации, метриками FID и анализом стабильности.

## Критерии и шкала оценивания (30 баллов)

| Критерий | Баллы |
|---|---|
| Корректность реализации Generator и Discriminator | 8 |

## Правила использования внешних ресурсов
- Реализация GAN должна быть выполнена самостоятельно (не использовать готовые реализации из репозиториев)
- Для расчёта FID допускается использование pytorch-fid или scipy
