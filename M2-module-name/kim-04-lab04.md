# КИМ-2.2. Лабораторная работа №4. Семантическая сегментация (U-Net, FCN)

## Назначение
Проверка способности применять архитектуры энкодер-декодер для задачи семантической сегментации изображений.

## Проверяемые результаты (дескрипторы)
- Применяет архитектуры U-Net и FCN для сегментации изображений
- Использует предобученные бэкбоны (backbone)
- Настраивает функции потерь (кросс-энтропия, Dice Loss)
- Оценивает качество сегментации по метрике IoU
- Разрабатывает и отлаживает прикладные решения на Python с использованием PyTorch, segmentation_models (PL-1.1)

## Условия выполнения
- Среда: Python 3.8+, Jupyter Notebook
- Библиотеки: PyTorch, segmentation_models_pytorch, OpenCV, NumPy, matplotlib

## Материалы задания
- **Датасет:** Cambridge-driving Labeled Video Database (CamVid) или Cityscapes (small subset)

### Задание
1. Загрузите датасет семантической сегментации
2. Реализуйте pipeline загрузки и аугментации данных
3. Загрузите предобученный бэкбон (ResNet-34, EfficientNet)
4. Реализуйте U-Net и FCN архитектуры с этим бэкбоном
5. Обучите модели с разными функциями потерь: Cross-Entropy, Dice Loss, Combo Loss
6. Оцените качество: IoU (Intersection over Union), Dice Coefficient
7. Визуализируйте результаты сегментации (исходное изображение, маска, предсказание)
8. Сравните U-Net и FCN по качеству и скорости

## Формат сдачи
Отчёт с результатами обучения, метриками, визуализациями и сравнительным анализом.

## Критерии и шкала оценивания (20 баллов)
