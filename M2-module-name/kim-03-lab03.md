# КИМ-2.1. Лабораторная работа №3. Разработка и обучение детектора объектов (YOLO)

## Назначение
Проверка способности применять современные одноэтапные детекторы объектов для решения практических задач.

## Проверяемые результаты (дескрипторы)
- Обучает детектор YOLO на пользовательском датасете
- Настраивает гиперпараметры обучения
- Оценивает качество детекции по метрике mAP
- Проводит инференс на видео и изображениях

## Условия выполнения
- Среда: Python 3.8+, Jupyter Notebook / скрипты
- Библиотеки: PyTorch, Ultralytics YOLO, OpenCV, NumPy

## Материалы задания
- **Датасет:** пользовательский (собрать и разметить 100–200 изображений) или подмножество COCO
- **Фреймворк:** Ultralytics YOLOv8 / YOLOv11

### Задание
1. Соберите или выберите датасет для детекции (не менее 3 классов объектов)
2. Разметьте изображения в формате YOLO (txt-файлы с координатами)
3. Обучите детектор YOLO на своём датасете
4. Настройте гиперпараметры: размер изображения, batch size, learning rate, количество эпох
5. Оцените качество: Precision, Recall, mAP@0.5, mAP@0.5:0.95
6. Проведите инференс на тестовых изображениях и видео
7. Визуализируйте результаты: bounding boxes с классами и уверенностью
8. Проанализируйте ошибки: false positive, false negative

## Формат сдачи
Отчёт (Jupyter Notebook / PDF) с результатами обучения, метриками и визуализациями + демонстрация работы детектора.

## Критерии и шкала оценивания (30 баллов)

| Критерий | Баллы |
|---|---|
| Корректность сбора и разметки датасета | 6 |
| Обучение модели и настройка гиперпараметров | 8 |
| Оценка качества (метрики, анализ ошибок) | 8 |
| Визуализация результатов и выводы | 8 |

## Правила использования внешних ресурсов
- Допускается Ultralytics YOLO, предобученные весы
- Датасет должен быть собран или выбран самостоятельно (не использовать готовые train/test split без модификаций)
