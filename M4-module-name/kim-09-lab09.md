# КИМ-4.2. Лабораторная работа №9. Распознавание текста на изображениях (CRNN)

## Назначение
Проверка способности применять свёрточно-рекуррентные сети для распознавания текста на изображениях.

## Проверяемые результаты (дескрипторы)
- Применяет CRNN (свёрточно-рекуррентную сеть) для распознавания текста
- Использует методы детекции текста (MSER, CRAFT)
- Оценивает точность распознавания (CER, WER)

## Условия выполнения
- Среда: Python 3.8+, Jupyter Notebook
- Библиотеки: PyTorch, torchvision, OpenCV, NumPy, matplotlib, Levenshtein

## Материалы задания
- **Датасет:** IIIT5K, SVHN (Street View House Numbers) или синтезированный датасет текста

### Задание
**Часть 1. Детекция текста (опционально — даётся готовый)**
- Примените MSER (Maximally Stable Extremal Regions) для выделения текстовых областей
- Используйте CRAFT или другой современный детектор текста (опционально)

**Часть 2. Распознавание текста (CRNN)**
1. Реализуйте CRNN архитектуру:
   - CNN backbone (VGG-like или ResNet) для извлечения признаков
   - RNN (BiLSTM) для последовательной обработки
   - CTC Loss (Connectionist Temporal Classification)
2. Подготовьте данные: синтезируйте или загрузите датасет изображений текста
3. Обучите CRNN на датасете
4. Оцените качество: CER (Character Error Rate), WER (Word Error Rate)
5. Протестируйте на реальных изображениях (фото вывесок, документов)
6. Проанализируйте ошибки: какие символы путаются чаще всего

## Формат сдачи
Отчёт с архитектурой CRNN, динамикой обучения, метриками CER/WER, примерами распознавания.
